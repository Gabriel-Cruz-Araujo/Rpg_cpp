## Descrição do Projeto
Este projeto de jogo RPG é um jogo simples baseado em console, desenvolvido para fins educacionais. Ele envolve mecânicas básicas de jogo, como criação de personagens, batalhas e gerenciamento de itens. O objetivo do projeto é demonstrar programação orientada a objetos e design básico de jogos em C++.

## Funcionalidades
- Criação de personagens
- Sistema de batalha
- Inventário e itens
- Salvar e carregar o estado do jogo

## Uso
Após abrir o projeto no Visual Studio, você pode modificar o código do jogo ou executá-lo diretamente no IDE.

## Compilar e Executar
Abra o arquivo de solução `Rpg.sln` no Visual Studio.
Selecione a configuração alvo (por exemplo, `Debug` ou `Release`) e a plataforma (por exemplo, `x64`).
Compile a solução usando o menu Build ou pressione `Ctrl+Shift+B`.
Execute o projeto usando `F5` ou através do menu Debug.

## Dependências
Este projeto requer:
-Visual Studio (recomendado para compilar e executar o projeto)
-Compilador C++ compatível com o padrão C++ usado no código
